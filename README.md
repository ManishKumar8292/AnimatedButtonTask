# Animated Button

## Project Description

This project is a simple animated button created using only **HTML and CSS**.
When the user hovers over the button, it becomes slightly bigger and rotates a little to create a smooth interactive effect.

## Technologies Used

- HTML5
- CSS3
- CSS Transitions
- CSS Transform

## What I Learned

- Creating smooth animations using `transition`
- Using `transform: scale()` and `rotate()`
- Improving UI interaction with hover effects
- Writing clean and reusable CSS

## How To Run

1. Download or clone the project.
2. Keep `index.html` and `style.css` in the same folder.
3. Open `index.html` in your browser.
